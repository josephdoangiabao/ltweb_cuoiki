Giải thích logic:

  • Số ngẫu nhiên được tạo bằng Math.floor(Math.random() * 101) + 50 (từ 50 đến 150).
  • So sánh input với số ngẫu nhiên, báo quá cao/thấp/đúng.
  • Đếm số lần thử, khi đúng thì hiện confetti bằng CSS.
  • Kiểm tra input để tránh lỗi nhập ngoài khoảng.
